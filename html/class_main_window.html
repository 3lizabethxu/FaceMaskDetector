MainWindow::MainWindow ( QWidget *  parent = nullptr)

Sets up the main window for the Qt interface, which uses a grid layout to organize the design.

There are 3 columns for the grid. The first column mainly features the video feed, the second column is a placeholder for spacing, and the third column features settings for the user to interact with the program (i.e. adjusting zoom, play/pause, and summary statistics)
